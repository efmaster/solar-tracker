# PWA Installation - Solarertrag Tracker

## 📱 Als App installieren

Die Solarertrag Tracker App kann als Progressive Web App (PWA) auf deinem Gerät installiert werden.

### Android (Chrome/Edge)

1. Öffne die App in Chrome oder Edge
2. Tippe auf das Menü (⋮) oben rechts
3. Wähle **"Zum Startbildschirm hinzufügen"** oder **"App installieren"**
4. Bestätige die Installation
5. Die App erscheint auf deinem Homescreen

### iOS (Safari)

1. Öffne die App in Safari
2. Tippe auf das Teilen-Symbol (□↑) unten
3. Scrolle und wähle **"Zum Home-Bildschirm"**
4. Bestätige mit **"Hinzufügen"**
5. Die App erscheint auf deinem Homescreen

### Desktop (Chrome/Edge)

1. Öffne die App in Chrome oder Edge
2. Klicke auf das Install-Symbol (⊕) in der Adressleiste
3. Oder: Menü → **"App installieren"**
4. Bestätige die Installation
5. Die App öffnet sich in einem eigenen Fenster

## ✨ Vorteile der PWA

- **Offline-Nutzung**: Daten bleiben auch ohne Internet verfügbar
- **Schneller Start**: App startet wie eine native App
- **Kein App Store**: Direkte Installation vom Browser
- **Automatische Updates**: Immer die neueste Version
- **Weniger Speicher**: Kleiner als native Apps
- **Homescreen-Icon**: Schneller Zugriff

## 🔧 Features

- Vollständige Offline-Funktionalität
- Lokale Datenspeicherung (SQLite)
- Push-Benachrichtigungen (optional)
- Responsive Design für alle Geräte
- Touch-optimierte Bedienung

## 📊 App-Icon

Das App-Icon zeigt:
- Sonne (Solarenergie)
- Solarpanel (grün)
- kWh-Label

## 🌐 Systemanforderungen

- **Android**: Chrome 90+ oder Edge 90+
- **iOS**: Safari 14+ (iOS 14+)
- **Desktop**: Chrome 90+, Edge 90+, oder Firefox 90+

## 🔄 Updates

Die App aktualisiert sich automatisch beim nächsten Start, wenn eine neue Version verfügbar ist.

## 🗑️ Deinstallation

### Android/iOS
Halte das App-Icon gedrückt → **"Deinstallieren"** oder **"Vom Homescreen entfernen"**

### Desktop
Rechtsklick auf App-Icon → **"Deinstallieren"** oder über Browser-Einstellungen

---

**Tipp**: Nach der Installation kannst du die App wie jede andere App verwenden - auch ohne Internetverbindung!
